# 🧠 交互式大脑解剖学图谱

面向**心理学研究生**的交互式 3D 大脑解剖学学习工具。

## 功能特色

- **3D 可交互大脑模型**：Three.js 构建，支持旋转、缩放、点击
- **21 个核心脑区**：覆盖前额叶系统、边缘系统、纹状体/奖赏系统、感觉系统、脑干/调节系统、白质连接
- **8 层深度信息**：解剖位置 → 细胞构筑 → 主要功能 → 连接组 → 神经递质 → 临床关联 → 关键研究 → 最新争论
- **心理学研究生级别深度**：亚核细节、精神疾病临床关联、研究范式、经典文献引用
- **实时搜索**：支持中英文/拼音搜索脑区（Ctrl+K 快捷键）
- **自测模式**：随机出题，在大脑上点击对应脑区
- **完全离线可用**：纯静态站点，无需服务器

## 技术架构

- 纯 HTML/CSS/JS（无框架，零构建步骤）
- Three.js（CDN 加载，WebGL 3D 渲染）
- GitHub Pages 部署

## 脑区覆盖

| 系统 | 脑区 |
|------|------|
| 前额叶系统 | dlPFC, vmPFC, OFC, ACC, 脑岛 |
| 边缘系统 | 杏仁核, 海马体 |
| 纹状体/奖赏 | NAc, VTA, 尾状核, 壳核 |
| 感觉系统 | 视觉皮层, 丘脑, S1 |
| 脑干/调节 | 下丘脑, 黑质, 中缝核, 蓝斑, PAG |
| 运动系统 | M1 |
| 白质/连接 | 胼胝体 |

## 本地运行

直接在浏览器中打开 `index.html` 即可。

或者在项目目录启动一个本地服务器：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

## 部署

推送到 GitHub 仓库的 `master` 分支，启用 GitHub Pages 即可。
